Materi Dasar Java

Java adalah bahasa pemrograman tingkat tinggi yang bersifat objek dan dilengkapi dengan sistem manajemen memori otomatis (garbage collection) Java memiliki dua komponen utama: Java Development Kit (JDK) untuk pengembangan dan Java Runtime Environment (JRE) untuk menjalankan aplikasi Java.Java didasarkan pada paradigma pemrograman berorientasi objek (OOP), di mana program dibangun dengan menggunakan objek dan kelas, dan masih banyak lagi.

Sejarah singkat Java :
  1. Java dikembangkan oleh James Gosling, Mike Sheridan, dan Patrick Naughton di Sun Microsystems pada awal 1990-an.
  2. Pendekatan Asal : Awalnya, Java dirancang sebagai bahasa pemrograman untuk perangkat elektronik konsumen, tetapi tidak lama kemudian beralih fokus ke pengembangan aplikasi perangkat lunak
  3. Peluncuran Publik : Java pertama kali diperkenalkan ke publik pada tahun 1995 bersamaan dengan peluncuran platform Java.
  4. Platform Independen : Salah satu fitur utama Java adalah kemampuannya untuk berjalan di berbagai platform tanpa memerlukan modifikasi ulang (platform-independent) karena menggunakan konsep "Write Once, Run Anywhere" (WORA).
  5. Evolusi dan Pengembangan : Sun Microsystems merilis versi-versi Java yang terus berkembang. Kemudian, Oracle Corporation mengakuisisi Sun Microsystems pada tahun 2010 dan menjadi pemelihara utama Java.
Java tetap menjadi salah satu bahasa pemrograman yang sangat populer di dunia, digunakan dalam berbagai konteks pengembangan termasuk perangkat lunak perusahaan, perangkat mobile, dan aplikasi web.
Sangat banyak syntax yang dapat digunakan dalam Java, bahkan kita bisa membuat GUI (Graphical User Interface) yang kemudian kita isikan sesuai dengan kegunaan dari setiap komponen dalam GUI tersebut.

Berikut contoh sederhana dari penggunaan syntax dalam Java :
public class Contoh {
public static void main(String[] args) {
int x = 10;
if (x > 5) {
System.out.println("x lebih besar dari 5");
} else {
System.out.println("x kurang dari atau sama dengan 5");
}
System.out.println("Program selesai.");
} }
