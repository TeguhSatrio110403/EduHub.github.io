Materi Dasar CSS

CSS adalah pengembang untuk memilih elemen HTML dan menerapkan gaya (style) ke elemen tersebut, sifatnya mengontrol berbagai aspek presentasi, seperti warna teks, ukuran font, tata letak, dan dekorasi elemen CSS memungkinkan pengembang menggunakan kelas dan ID untuk memberikan gaya khusus kepada elemen atau kelompok elemen tertentu, CSS memungkinkan pembuatan desain responsif dengan menggunakan media queries, di mana tata letak dan gaya dapat disesuaikan berdasarkan karakteristik perangkat atau tampilan.

Sejarah singkat CSS :
  1. Asal Mula : CSS (Cascading Style Sheets) dikembangkan oleh Hakon Wium Lie dan Bert Bos. CSS pertama kali diusulkan pada tahun 1994 sebagai respons terhadap kebutuhan untuk pemisahan antara struktur HTML dan presentasi tata letak.
  2. Pemisahan Presentasi dan Struktur : Tujuan utama CSS adalah memisahkan struktur dokumen HTML (HTML markup) dari presentasi atau tata letaknya. Sebelum adanya CSS, tata letak dikendalikan langsung di dalam elemen HTML.
  3. Standardisasi : CSS Level 1 menjadi rekomendasi dari W3C (World Wide Web Consortium) pada tahun 1996. Ini memulai langkah-langkah menuju standardisasi untuk pemrograman antarmuka web.
  4. Evolusi CSS : CSS mengalami evolusi dengan diperkenalkannya CSS Level 2 pada tahun 1998 dan kemudian CSS Level 3 yang masih terus berkembang hingga sekarang. Setiap level membawa perbaikan dan penambahan fitur.
CSS adalah bahasa gaya yang sangat penting dalam pengembangan web modern, memungkinkan pengembang untuk menciptakan tampilan dan tata letak yang menarik dan responsif.
