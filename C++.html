Materi Dasar C++

C++ adalah bahasa pemrograman umum yang mendukung pemrograman prosedural dan berorientasi objek. Hal ini membuatnya sangat fleksibel dan dapat digunakan dalam berbagai konteks pengembangan, C++ memungkinkan pengembang untuk menggunakan konsep OOP seperti kelas, objek, pewarisan, enkapsulasi, dan polimorfisme. C++ memberikan kontrol langsung terhadap alokasi dan dealokasi memori, memberikan keleluasaan kepada pengembang untuk mengelola sumber daya secara manual. Dikenal dengan kinerjanya yang tinggi dan dapat digunakan dalam pengembangan perangkat lunak yang membutuhkan efisiensi tinggi, seperti pengembangan sistem operasi atau perangkat lunak embedded.

Sejarah singkat C++ :
  1. Asal Mula : C++ dikembangkan oleh Bjarne Stroustrup di Bell Labs pada awal 1980-an sebagai perluasan dari bahasa pemrograman C. Stroustrup merancang C++ dengan tujuan menggabungkan fleksibilitas C dengan fitur-fitur pemrograman berorientasi objek.
  2. Pemrograman Berorientasi Objek : Salah satu kontribusi utama C++ adalah pendekatannya yang berorientasi objek, memungkinkan pengembang untuk menggunakan konsep seperti enkapsulasi, pewarisan, dan polimorfisme.
  3. Standarisasi : C++ pertama kali diresmikan sebagai standar internasional pada tahun 1998 oleh Organisasi Standar Internasional (ISO). Standar ini dikenal sebagai "C++98".
  4. Evolusi C++ : C++ terus mengalami evolusi, dengan penyempurnaan dan penambahan fitur melalui standar berikutnya seperti C++03, C++11, C++14, C++17, dan seterusnya. Setiap standar memperkenalkan fitur-fitur baru dan peningkatan kinerja.
C++ tetap menjadi salah satu bahasa pemrograman yang populer dan memiliki peran yang signifikan dalam pengembangan perangkat lunak di berbagai domain.
Dalam C++ terdapat banyak elemen sintaks yang dapat digunakan, misalnya :
  • #include: Digunakan sebagai file header atau library.
  • using namespace std; : Menggunakan namespace dari library standar.
  • int main() : Fungsi utama yang dijalankan saat program dimulai.
  • cout : Mengeluarkan data ke layar.
  • cin : Memasukan inputan dari luar.
  • endl : Pergantian baris.

Contoh Sederhana :
#include
using namespace std;
int main() {
int x = 10;
if (x > 5) {
cout << "x lebih besar dari 5" << endl;
} else {
cout << "x tidak lebih besar dari 5" << endl;
}
return 0;
}
Setelah file di eksekusi, maka akan menghasilkan output berupa : x lebih besar dari 5. Hal ini terjadi karena x bernilai 10, sehingga akan membaca parameter yang pertama.

Masih banyak hal yang dapat di pelajari, ini hanya perkenalan awal mengenai bahasa pemrograman C++. Untuk mempermuda membuat kode dengan C++, kita menggunakan Compiler, biasanya yang paling banyak digunakan adalah Dev C++, Sublime, Visual Studio Code.
