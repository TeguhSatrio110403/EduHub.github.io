Dasar HTML : Mengenal Tag dalam HTML

HTML (Hypertext Markup Language) adalah bahasa pemrograman yang digunakan untuk membuat struktur dan tampilan halaman web. Ini mengandung elemen-elemen yang mendefinisikan berbagai komponen di halaman web, seperti teks, gambar, tautan, dan formulir. HTML menggunakan tag-tag untuk mengatur dan memformat konten web, dan browser web kemudian menginterpretasikan kode HTML ini untuk menampilkan halaman web kepada pengguna.

Elemen HTML terbagi menjadi bagian HEAD (berisi title, meta tag, CSS code, dan JavaScript code) dan BODY (berisi konten HTML).

Tag dasar HTML mencakup :
  • Elemen HEAD dan BODY.
  • Elemen HEAD memiliki tag TITTLE, BASE, LINK, dan META.
  • Elemen BODY menyimpan konten HTML seperti teks, hyperlink, gambar, dan tabel.
  • Browser memproses script HTML untuk menampilkan halaman web.
  • Format teks menggunakan tag FONT, b, strong, em, i, small, big, sub, sup, u, s, dan lainnya.
  • Karakter khusus seperti ©, ®, ™, &, non-breaking space, «, », dan lainnya dapat ditampilkan menggunakan HTML entities.
  • Meta tag seperti : meta charset="utf-8". Digunakan untuk menentukan karakter encoding.
  • Gambar ditampilkan menggunakan tag IMG dengan atribut src, width, height, alt, align, title, dan border.
  • Suara dapat ditampilkan menggunakan tag EMBED atau BGSOUND.
  • Video dapat ditampilkan menggunakan tag EMBED dengan atribut src, width, height, dan type.
